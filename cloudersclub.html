Clouders Club was formed on January 17th 2015. The aim of the club was to engage students in the field of Cloud Computing focusing on Cloud Apps like Google Cloud Platform, Amazon Web Services & Microsoft Azure cloud services.

" Cloud Computing", by definition, refers to the on-demand delivery of IT resources and applications via the Internet with pay-as-you-go pricing.

Programming Club has acquired Clouders Club.
